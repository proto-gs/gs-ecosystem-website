Вопрос — Ответ FAQ — Questions & Answers

1. Что представляют собой приложения GS.HTTP? 1. What are GS.HTTP applications?

➔
Это специализированные мобильные утилиты, предназначенные для быстрой проверки HTTP-запросов, анализа ответов серверов и оперативной диагностики веб-сайтов. These are specialized mobile utilities designed for quick verification of HTTP requests, analysis of server responses, and prompt website diagnostics.

2. В чем главное различие между полной версией GS.HTTP и облегченной GS.HTTP.Lite? 2. What is the main difference between the full version of GS.HTTP and the lightweight GS.HTTP.Lite?

➔
Версия Lite оптимизирована для работы на слабых устройствах и включает только базовые функции проверки статус-кодов и заголовков ответов. The Lite version is optimized for low-end devices and includes only basic features for checking status codes and response headers.

3. Как с помощью приложения узнать HTTP-статус веб-ресурса? 3. How do I find out the HTTP status of a web resource using the app?

➔
Достаточно ввести URL-адрес интересующего вас сайта в поисковую строку на главном экране приложения и нажать кнопку «ПРОВЕРИТЬ» для отправки запроса. Simply enter the URL of the site you are interested in into the search bar on the main screen of the app and press the "CHECK" button to send the request.

4. Можно ли в GS.HTTP анализировать технические заголовки (Headers)? 4. Is it possible to analyze technical headers in GS.HTTP?

➔
Да, после отправки запроса и получения ответа от сервера приложение выводит полный список всех технических заголовков для их детального анализа разработчиком или администратором. Yes, after sending a request and receiving a response from the server, the app displays a complete list of all technical headers for detailed analysis by a developer or administrator.

5. Поддерживают ли утилиты форматирование и чтение JSON-данных? 5. Do the utilities support JSON data formatting and reading?

➔
Полная версия GS.HTTP позволяет просматривать и автоматически форматировать сырые JSON-ответы в структурированный, древовидный и удобный для чтения вид. The full version of GS.HTTP allows you to view and automatically format raw JSON responses into a structured, tree-like, and easy-to-read format.

6. Могу ли я проверить файлы Cookies, которые устанавливает сайт? 6. Can I check the Cookies set by a website?

➔
Да, полноценная функция просмотра, фильтрации и анализа файлов Cookies, возвращаемых веб-ресурсами, интегрирована в интерфейс приложения. Yes, a fully functional feature for viewing, filtering, and analyzing Cookies returned by web resources is integrated into the application interface.

7. Требуется ли регистрация или создание аккаунта для использования утилит? 7. Is registration or account creation required to use the utilities?

➔
Нет, оба приложения полностью минималистичны. Весь диагностический функционал доступен пользователю сразу после запуска без ввода личных данных и авторизации. No, both applications are completely minimalist. All diagnostic functionality is available to the user immediately upon launch, without entering personal data or authorization.

8. Собирают ли приложения GS.HTTP историю проверяемых мной сайтов? 8. Do GS.HTTP applications collect the history of the websites I check?

➔
Программы работают исключительно локально на вашем устройстве и не занимаются скрытым сбором, хранением или передачей персональной информации третьим лицам. The programs work exclusively locally on your device and do not engage in hidden collection, storage, or transmission of personal information to third parties.

9. На каких устройствах и операционных системах работают данные приложения? 9. On which devices and operating systems do these applications work?

➔
Данные утилиты разработаны исключительно для мобильной операционной системы Android и предназначены только для установки на Android-смартфоны. These utilities are developed exclusively for the Android mobile operating system and are intended only for installation on Android smartphones.

10. Где можно посмотреть всю техническую информацию и сведения о приложении? 10. Where can I view all technical details and information about the application?

➔
Вся актуальная техническая информация, номер текущей сборки и правовые сведения находятся в кнопке «Информация» в главном меню приложения. All up-to-date technical information, the current build number, and legal information can be found under the "Information" button in the application's main menu.
