Политика конфиденциальности GS.Monitor

Политика конфиденциальности десктопного приложения GS.Monitor по требованиям Роскомнадзора и Федерального закона № 152-ФЗ "О персональных данных" включает в себя план из 7 пунктов:
1. Общие положения и соответствие законодательству
2. Лицензия MIT и аудит открытого исходного кода
3. Категории данных и заявление о локальном хранении
4. Цели обработки и функция поиска сайтов
5. Отсутствие телеметрии и рекламных трекеров
6. Права пользователей и ответственность за сбор данных
7. Контактная информация

1. Общие положения и соответствие законодательству

Настоящая политика конфиденциальности определяет порядок обработки информации пользователей десктопного приложения «GS.Monitor» (далее — «приложение»).
Данный документ разработан в соответствии с Федеральным законом РФ № 152-ФЗ «О персональных данных», а также принципами международных стандартов защиты данных (GDPR в Европейском Союзе и CCPA в США) для обеспечения полной прозрачности.

2. Лицензия MIT и аудит открытого исходного кода

Исходный код приложения является полностью открытым и распространяется на условиях свободной лицензии MIT License.
Публичный статус кода позволяет любому пользователю или независимому эксперту лично проверить программное обеспечение, чтобы убедиться в абсолютном отсутствии скрытых алгоритмов отслеживания, телеметрии или несанкционированного сбора информации.

3. Категории данных и заявление о локальном хранении

3.1. Персональные данные: Приложение не мобирает, не запрашивает и не хранит никакие персональные данные (такие как имена, адреса электронной почты, номера телефонов или IP-адреса) на каких-либо внешних серверах, принадлежащих разработчику.

3.2. Рабочие (технические) данные: Для выполнения функций HTTP-клиента и парсинга приложение обрабатывает вводимые пользователем параметры: целевые URL-адреса, HTTP-методы (GET, POST, PUT, DELETE), заголовки запросов (Headers), API-ключи, токены авторизации и ключевые слова для поиска сайтов.

3.3 Локальный статус: Все рабочие данные, история запросов и результаты поиска хранятся исключительно на локальном жёстком диске компьютера пользователя.
Разработчик имеет нулевой технический доступ к вашим данным.

4. Цели обработки и функция поиска сайтов

Обработка технических данных происходит исключительно на стороне клиента для выполнения команд сетевых протоколов по прямому запросу пользователя.
При использовании функции поиска сайтов по ключевому слову, введенная строка передается напрямую на API публичных поисковых систем без участия серверов разработчика.
Сетевой трафик идет напрямую с сетевой карты компьютера на целевой URL.

5. Отсутствие телеметрии и рекламных трекеров

Приложение создано на фреймворке Kotlin/Compose Jetpack и не содержит встроенных сторонних модулей телеметрии, инструментов отправки отчетов о сбоях (таких как Firebase Crashlytics, Sentry) или скрытых рекламных SDK.

6. Права пользователей и ответственность за сбор данных

6.1. Пользователь сохраняет абсолютный контроль над своей цифровой информацией.
Вы можете мгновенно очистить весь журнал запросов и историю поиска сайтов через настройки приложения или полностью удалить программное обеспечение с жесткого диска.

6.2 Пользователь самостоятельно несет 100% юридическую ответственность за всю сетевую активность, легитимность сбора данных (веб-скрейпинга) с сайтов, обнаруженных с помощью приложения, и обязан соблюдать правила файлов robots.txt целевых ресурсов.

7. Контактная информация

GS.Monitor Privacy Policy

The GS.Monitor desktop app's privacy policy, in accordance with EU GDPR (General Data Protection Regulation) and the California Consumer Privacy Act (CCPA), includes a 7-point plan:
[EN]
1. General provisions & compliance
2. MIT License & open-source audit
3. Data categories & local staorage declaration
4. Purposes of processing & website search transmission
5. No telemetry or advertising trackers
6. User rights and data scraping liability
7. Contact information

1. General provisions & compliance

This Privacy Policy governs the processing of user information within the "GS.Monitor" desktop application (hereinafter — the "application").
This document complies with global privacy protection laws, including the EU GDPR and California CCPA, ensuring full transparency for client-side software.

2. MIT License & open-source audit

The Application’s source code is fully open-source and distributed under the MIT License.
The public status of the code allows any User or independent security expert to personally inspect the software to verify the absolute absence of hidden tracking, telemetry, or data harvesting algorithms.

3. Data categories & local staorage declaration

3.1. Personal Data: The application does not collect, request, or store any Personal Data (such as names, emails, phone numbers, or IP addresses) on any external servers owned by the developer.
3.2. Operational Data: To function as an HTTP client and scraping utility, the application processes user-inputted parameters: target URLs, HTTP methods (GET, POST, PUT, DELETE), custom Request Headers, API keys, Bearer tokens, and website search keywords.
3.3. Zero-Server Architecture: All operational data, request histories, and discovered URLs are stored exclusively on your local hard drive.
The developer has zero technical access to your data.

4. Purposes of processing & website search transmission

Technical data processing occurs solely client-side to execute network protocol commands at the User's explicit request.
When utilizing the website search feature, the inputted search string is sent directly to public search engine APIs to populate the output log.
Network traffic is transmitted directly from the User's network interface card to the target URL without passing through any intermediate servers of the developer.

5. No telemetry or advertising trackers

The Application is built using the Flet framework and does not contain hidden third-party telemetry modules, crash reporting tools (such as Firebase, Sentry), or advertising SDKs.

6. User rights and data scraping liability

6.1. The User maintains absolute ownership over their data footprint. You may wipe your complete request log and search history instantly using the Application's settings or by uninstalling the software from your PC.
6.2. The User assumes 100% legal liability for all network activity, including the legality of harvesting data (web scraping) from third-party websites discovered through the Application, and agrees to comply with target robots.txt files.

7. Contact information

For questions regarding your privacy while using the application, please contact the Developer at: georgsmerdov@outlook.com.
