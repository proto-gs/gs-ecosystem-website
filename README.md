# GS | Мои проекты

Официальный сайт-визитка разработчика **G. Smerdov**, объединяющий информацию о мобильных и десктопных приложениях, дизайне интерфейсов и новостях проектов.

## Ссылка на проект
Сайт опубликован и доступен по адресу: [https://gs-ht.ru]

## Ключевые проекты на сайте
* **GS.HTTP** — инструмент для глубокого анализа HTTP-трафика на Android.
* **GS.HTTP Lite** — легковесная версия приложения для базового анализа трафика.
* **GS.Monitor** — продвинутый мониторинг системных ресурсов (в разработке).

## Технологический стек
* **HTML5** — семантическая структура страниц.
* **CSS3** — кастомный адаптивный дизайн с поддержкой тем оформления.
* **JavaScript** — интерактивные элементы и логика переключения экранов.

##  Структура репозитория
* `/index.html` — главная страница портфолио.
* `/privacy.html` — политика конфиденциальности для публикуемых приложений.
* `/robots.txt` — конфигурация для поисковых систем.
* Скриншоты и графические ресурсы (логотипы RuStore, RuMarket, Telegram).
