# GS | Ecosystem

Официальный сайт-визитка разработчика **Георгий Смердов (ГС Экосистем)**, объединяющий информацию о мобильных и десктопных приложениях, документации и проекта в целом.

## Ссылка на проект
Сайт опубликован и доступен по адресу: [https://gs-ht.ru]

## Ключевые проекты на сайте
* **GS.HTTP** — Инструмент для глубокого анализа HTTP-трафика на Android.
* **GS.Monitor** — Мощный инструмент анализа HTTP-трафика для ПК.

## Технологический стек
* **HTML5** — семантическая структура страниц.
* **CSS3** — кастомный адаптивный дизайн с поддержкой тем оформления.
* **JavaScript** — Смена языков EN -> RU -> EN.

##  Структура репозитория
* `/index.html` - главная страница проекта.
* `/documentation.html` - Документации проектов.
* `/apps.html` - Приложения проекта.
* `/privacy-policy.html` - Политика конфиденциальности сайта.
* `/robots.txt` - Конфигурация для поисковых систем.
* Скриншоты приложений для документации хранятся в папках (Documentation_GS.HTTP, Documentation_GS.Monitor).
